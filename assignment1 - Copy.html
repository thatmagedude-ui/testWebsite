Structure of HTML

Your website is made up of many elements. Each element is either inside another element (like how < li > tags go inside <ol> tags) or after another element (like how this <p> element is after an <h1> element)

Attributes vs CSS Properties

Attributes include things like src, id, and href. CSS properties are things like color and font-size. REMEMBER, PROPERTIES ARE NOT ATTRIBUTES. THEY FOLLOW DIFFERENT RULES.

What is CSS?

CSS stands for Cascading Style Sheets.
Cascading because there are levels of specificity. If you have a color defined both inline and embedded, the inline styling will decide the color
Style because we use style properties (NOT ATTRIBUTES) for things like inline and embedded css.
Sheets because for big projects, your embedded or external stylesheets can take up entire pages to make your website look exactly the way you need it to look.
